# Exp - 5 STUDENT ENROLLMENT SYSTEM

# AIM:
To draw UML Diagrams for the Student Enrollment System
# SRS (Procedure):
◉ Use Case Diagram

◉ Class Diagram
◉ Activity Diagram
◉ Package Diagram
◉ Communication Diagram
◉ Sequence Diagram
1.Identify Actors: Define the main entities such as Student, Admin, and Course.
2.Use Case Modeling: Create use case diagrams showing interactions like registration, enrollment, and course management.
3.Class and Sequence Diagrams: Draw class diagrams to represent relationships between objects and sequence diagrams to show data flow
4.Activity and State Diagrams: Represent the workflow of student enrollment and transitions between system states.
